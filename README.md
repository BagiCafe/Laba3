# LABA3 ( Алгоритмический мини-пакет)

## Установка и запуск

### Запуск Лабы
```bash

git clone <https://github.com/BagiCafe/Laba3.git>

# Установите pytest для тестов
pip install pytest

# Запуск демонстрации всех функций
python main.py
```
## Функционал

### Функции сортировок
1) ```bubble_sort``` ( Пузырьковая сортировка)
   - Скорость сортировки : O(n²) в худшем и среднем случае, O(n) в лучшем

3) ```quick_sort``` ( Быстрая сортировка)
   - Скорость сортировки : O(n log n) в среднем, O(n²) в худшем случае

3) ```counting_sort``` ( Сортировка подсчетом)
   - Скорость сортировки : O(n + k) где k - диапазон значений

3) ```radix_sort``` ( Поразрядная сортировка)
   - Скорость сортировки : O(nk) где k - количество разрядов

3) ```bucket_sort``` ( Блочная сортировка)
   - Скорость сортировки : O(n + k) в среднем, O(n²) в худшем случае

3) ```heap_sort``` ( Пирамидальная сортировка)
   - Скорость сортировки : O(n log n) во всех случаях

### Структуры данных
1) ```Stack``` ( Стек)
   - Поддерживает такие функции как: ```push```, ```pop```, ```peek```, ```is_empty```, ```__len__```, ```min```
2) ```Queue``` ( Очередь)
   - Поддерживает такие функции как: ```enqueue```, ```dequeue```, ```front```, ```is_empty```, ```__len__```

### Факториал и Фибоначчи
1) Факториал
   - ```factorial(n)``` - Итеративная функция
   - ```factorial_recursive(n)``` - Рекурсивная функция
2) Фибоначчи
   - ```fibo(n)``` - Итеративная функция
   - ```fibo_recursive(n)``` - Рекурсивная функция

### Генерация тест-кейсов
#### Функции:
- ```rand_int_array``` - Генерация случайного массива целых чисел
- ```nearly_sorted``` - Генерация почти отсортированного массива
- ```many_duplicates``` - Генерация массива с большим количеством дубликатов
- ```reverse_sorted``` - Генерация обратного отсортированного массива
- ```rand_float_array``` - Генерация случайного массива дробных чисел

### Бенчмарк ( Система для сравнения производительности алгоритмов)
#### Функции:
- ```timer_once``` - Измеряет время выполнения функции
- ```benchmark_sorts``` - Сравнивает производительности алгоритмов сортировки

## Тестирование
Для запуска всех тестов необходимо ввести в консоль:
```bash
python -m pytest tests/ -v
```

Для запуска определенного теста ввидите в консоль:
```bash
python -m pytest tests/<file_name_test> -v
```
Название всех тестов:
- test_sorting.py
- test_stack_queue.py
- test_fibo_fact.py
- test_generators.py
- test_benchmark.py
